# 南雅医院企业中后台管理系统

## 项目简介

南雅医院企业中后台管理系统是一个基于Vue 3 + TypeScript + Element Plus开发的现代化企业管理系统，主要用于医院内部的档案管理和知识库管理。系统具有美观的界面设计、流畅的用户体验和完善的功能模块。

### 主要功能

- **用户认证**：登录、注册功能，支持记住密码
- **档案管理**：档案列表、档案详情、创建档案、编辑档案、档案分类管理
- **知识库管理**：知识文章列表、文章详情、创建文章、编辑文章、知识分类管理
- **系统概览**：数据统计、最近操作记录

## 技术栈

### 前端技术

- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **UI组件库**：Element Plus
- **路由管理**：Vue Router
- **状态管理**：Pinia
- **HTTP客户端**：Axios
- **CSS框架**：Tailwind CSS
- **图标库**：Element Plus Icons

### 后端技术

- **框架**：FastAPI
- **ORM**：SQLModel
- **数据库**：SQLite
- **认证**：JWT

## 安装步骤

### 前置要求

- Node.js 16.x 或更高版本
- npm 7.x 或更高版本

### 安装依赖

```bash
# 进入项目目录
cd frontend

# 安装依赖
npm install
```

### 启动开发服务器

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 使用说明

### 访问项目

开发服务器启动后，可通过以下地址访问项目：

```
http://localhost:5173/
```

### 登录系统

1. 在登录页面输入用户名和密码
2. 点击"登录"按钮进入系统
3. 勾选"记住我"可在下次访问时自动填充用户名

### 系统功能

#### 首页
- 查看系统概览数据
- 查看最近操作记录

#### 档案管理
- **档案列表**：查看所有档案，支持筛选和搜索
- **档案详情**：查看档案的详细信息
- **创建档案**：添加新的档案记录
- **编辑档案**：修改现有档案信息
- **分类管理**：管理档案的分类

#### 知识库管理
- **文章列表**：查看所有知识文章，支持筛选和搜索
- **文章详情**：查看文章的详细内容
- **创建文章**：添加新的知识文章
- **编辑文章**：修改现有文章内容
- **分类管理**：管理知识文章的分类

## 响应式设计

系统采用响应式设计，适配不同屏幕尺寸：

- **桌面端**：完整功能展示，多列布局
- **平板端**：适配中等屏幕，调整布局为两列
- **移动端**：适配小屏幕，单列布局，优化触摸操作

## 开发指南

### 代码规范

- 使用TypeScript进行类型检查
- 遵循Vue3组合式API的开发风格
- 使用ESLint进行代码质量检查
- 代码注释清晰，便于维护

### 组件开发

- 组件化开发，提高代码复用性
- 合理使用Props和Events进行组件通信
- 使用Pinia进行状态管理，避免组件间状态混乱

### 样式开发

- 使用Tailwind CSS进行样式开发
- 结合Element Plus的主题配置
- 保持一致的设计风格和视觉语言

## 项目特色

1. **现代化界面**：采用玻璃态设计、渐变背景和流畅动画，提供美观的用户体验
2. **完整功能**：包含用户认证、档案管理、知识库管理等核心功能
3. **响应式设计**：适配各种屏幕尺寸，提供一致的用户体验
4. **安全可靠**：使用JWT进行身份认证，保护用户数据安全
5. **易于扩展**：模块化设计，便于添加新功能和修改现有功能
